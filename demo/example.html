  ngContentEditable - Edit Mode

Example using ngContentEditable

This has editable regions!

You can go ahead and modify any regions on this page where editing is allowed.

You can modify text in these regions, or drag and drop assets from elsewhere on the page (where allowed). These assets might be images, or as you'll see, custom directives which will behave differently when placed within editable regions - just with a few lines of code.

Support for dragging files from your local system is allowed. You can even save your changes, in this example, with a little help from LocalStorage.

Follow the steps outlined below to build a simple application with editable page content. We'll build a page with Bootstrap for style and probably throw in a bit of jQuery just to cut corners for demonstration purposes.
